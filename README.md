# DATABASE MONA

DATABASE MONA è l'archivio operativo dell'universo H.O.T.H. (Heroes of the Hills).

Questo repository contiene il template grafico, le linee guida editoriali e tutti i dossier ufficiali prodotti dall'organizzazione.

---

## Obiettivi

- mantenere un'identità grafica coerente;
- documentare fenomeni, tecnologie, luoghi e minerali;
- utilizzare un linguaggio scientifico e operativo;
- garantire che tutti i dossier derivino da un unico Master Template.
